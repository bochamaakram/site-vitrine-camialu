## 📂 Project Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/company-showcase.git
   ```
2. Navigate to the project folder:
   ```sh
   cd company-showcase
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
